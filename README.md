# 💳 Mini ATM Machine (JavaScript)

## 📌 Project Description

This is a simple **Mini ATM Machine** built using JavaScript. It simulates basic ATM operations like checking balance, depositing money, and withdrawing money through user input.

---

## 🚀 Features

* Check account balance
* Deposit money
* Withdraw money
* Input validation (no invalid transactions)
* Simple and user-friendly logic

---

## 🛠️ Technologies Used

* JavaScript
* HTML (for interface and to see output using console in browser)

---

## 📂 How to Run

1. Clone this repository
2. Open the project folder
3. Run the file in browser or Node.js

---

## 💡 Example Operations

* Enter choice (Deposit / Withdraw / Check Balance)
* Enter amount
* Get updated balance instantly

---

## ⚠️ Limitations

* No real banking system (simulation only)
* No database integration
* Works on static data
